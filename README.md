# 🛡️ Raksha – Women Safety Android Application

## Overview

Raksha is an Android-based Women Safety Application designed to provide immediate assistance during emergency situations. The application enables users to send SOS alerts, share live location, notify emergency contacts, and quickly request help using multiple activation methods.

The goal of Raksha is to improve personal safety by reducing the time required to contact trusted people during emergencies.

---

# Features

* 🚨 One-Tap SOS Emergency Button
* 📍 Live Location Sharing
* 📱 Emergency Contact Management
* 🔐 Firebase Authentication
* 🗺️ Google Maps Integration
* 📩 SMS Alert with Current Location
* 📞 Emergency Calling
* 🎤 Voice Command Activation
* 📳 Shake Detection for SOS Trigger
* 🔔 Local Notifications
* 💾 Room Database for Local Storage
* 🎨 Modern Jetpack Compose UI

# Application Workflow

1. User Login / Registration
2. Home Dashboard
3. Add Emergency Contacts
4. Enable Location Services
5. Trigger SOS using:

   * SOS Button
   * Voice Command
   * Shake Detection
6. Live Location Shared
7. SMS Sent to Emergency Contacts
8. Emergency Call Initiated
